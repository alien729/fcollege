Desenvolvimento de Sistemas

Você já utilizou os diversos benefícios da Internet, já navegou em jornais, revistas, grupos de discussão e compras pela Internet. O curso de Desenvolvimento de Sistemas I irá mostrar COMO desenvolver sites estáticos e dinâmicos que poderão ser hospedados em servidores espalhados pelo mundo. Durante o curso será estudado:

  • HTML para elaboração de páginas estáticas.
  • CSS folhas de estilo em cascata para padronizar o leiaute das páginas elaboradas.
  • Bootstrap para tornar suas páginas visualmente mais atraentes e responsivas.
  • PHP estruturado que tornará suas páginas dinâmicas (conexão com banco de dados, escrita e leitura em arquivo texto...).
  • PHP orientado a objetos que irá facilitar a programação e tornar seus códigos mais limpos.
  • Laravel que é um framework muito utilizado na internet para o desenvolvimento de sistemas WEB
  • JavaScript que será utilizado para tornar suas páginas mais dinâmicas.
  • XAMPP é um software livre, servidor independente de plataforma, que possui a base de dados MySQL, o servidor web Apache e os interpretadores para linguagens de script: PHP e Perl.
